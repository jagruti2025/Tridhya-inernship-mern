# Library Management System

A simple frontend mini project that helps manage a collection of books. This project performs basic **CRUD (Create, Read, Update, Delete)** operations using plain JavaScript, along with search, filter, and sort features.

---

## About the Project

This is a beginner-friendly Library Management System built using HTML, Tailwind CSS, and JavaScript. It allows users to add new books, view them in a table, edit book details, and delete books when needed. The project also tracks whether a book is available or borrowed and shows simple statistics like total books, available books, and categories.

All data is saved in the browser using `localStorage`, so the book list stays even after the page is refreshed.

---

## Features

- Add new books with title, author, category, ISBN, year, and status
- View all books in a table
- Edit and update book details
- Delete books with a confirmation popup
- Search books by title
- Filter books by category and status
- Sort books by title or year
- Borrow and return books
- Live statistics for total, available, and borrowed books
- Data saved using localStorage
- Responsive design for mobile and desktop
